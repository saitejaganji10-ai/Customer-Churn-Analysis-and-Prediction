# Customer Churn Analysis and Prediction

## 📌 Project Overview

This project analyzes customer behavior and predicts customer churn using Machine Learning techniques.

Developed during the **SaiKet Systems Data Analysis Internship**, the project performs:

- Data Cleaning and Preprocessing
- Exploratory Data Analysis (EDA)
- Customer Segmentation
- Correlation Analysis
- Customer Lifetime Value Analysis
- Machine Learning Modeling
- Customer Retention Strategy Recommendations

---

## 📊 Dataset Information

- Total Customers: 7043
- Features: 21
- Target Variable: Churn

### Confusion Matrix

|-------------------|-----------|--------|
| No Churn | 934 | 102 |

---

## 🔥 Key Findings

- 73.5% customers retained their services.
- Month-to-month contracts showed the highest churn.
- Short-tenure customers were more likely to churn.
- Higher monthly charges increased churn probability.
- Fiber optic users exhibited higher churn rates.
- Electronic check users showed higher churn rates.
- Long-term contracts improved retention.
